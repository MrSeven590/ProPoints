# 工分录入页面日期步进器弹窗

## Goal
在工分录入页面的基本信息一栏的日期字段，点击后弹出窗口，使用步进器对日期进行调节。

## Requirements
- 日期字段可点击
- 点击后弹出模态窗口
- 弹窗中显示当前日期
- 提供 +1 天和 -1 天的步进器按钮
- 可以直接显示调整后的日期
- 点击确认后更新日期字段
- 点击取消或遮罩关闭弹窗

## Acceptance Criteria
- [ ] 日期字段添加点击事件
- [ ] 弹窗样式符合项目规范（参考现有弹窗实现）
- [ ] 步进器功能正常（+1/-1 天）
- [ ] 日期格式保持一致（YYYY-MM-DD）
- [ ] 确认后正确更新 sessionDate
- [ ] 取消后不更新日期
- [ ] 遮罩点击可关闭弹窗

## Technical Notes
- 参考 `biz-score-input` 组件的步进器实现
- 参考 `biz-bin-card` 组件的弹窗实现
- 使用 UTS 的日期处理方法（避免使用 Date 对象的不支持方法）
- 遵循 uni-app X 的样式约束（仅 flex 布局）
